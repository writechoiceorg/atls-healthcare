id integer
A unique identifier for the pickup request.
Example: 134

requestDateTime string (date-time)
The date and time when the pickup request was made.
Example: 2024-07-01T16:31:40.9603962+00:00

pickupTime integer
The scheduled time for the pickup in military format (e.g., 700 for 7:00 AM).
Example: 700

closingTime integer
The scheduled closing time in military format (e.g., 1700 for 5:00 PM).
Example: 1700

branchEmail string
The email address of the branch handling the pickup.
Example: HYZ_Pickups@ats.ca

customerEmail string
The email address of the customer who requested the pickup.
Example: john.doe@email.com

shipmentDetail object

Details of the shipment included in the pickup request.

pieces integer
The number of pieces in the shipment.
Example: 10

weight number
The total weight of the shipment in pounds.
Example: 150.5
