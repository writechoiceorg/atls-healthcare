Frontend App

The ACS Frontend App provides an intuitive, web-based interface for shipment management.

When to Use the Frontend App

  • Ease of Use: Simple, user-friendly interface without coding.
  • Small to Medium Volume: Ideal for occasional or moderate volume shipments.
  • Quick Setup: Start quickly without integration complexities.

Integrating with the API

The ATS API offers a flexible and powerful way to integrate shipping functionalities directly into your applications.

When to Use the API

  • Automation: Automate shipping processes and integrate with existing systems.
  • High Volume: Ideal for large-volume users requiring efficient batch processing.
  • Customization: Tailor shipping workflows to your business needs.
